# KCD Suisse Romande 2026

Kubernetes Community Days (KCD) Suisse Romande 2026 is a conference on 09-10 December 2026 at CERN. 
It is organized by the [Association Cloud Native Suisse Romande](https://cloud-native-romandy.ch),
with support from many sponsors and community partners.

This repository contains information and assets for the event until the main site is live.
Later, it will serve as an archive.
